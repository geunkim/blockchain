# Identity (신원)

이 저장소는 분산 웹(decentralized web) 환경을 위한 Identity와 관련한 정보를 가지는 장입니다.

* Self-Sovereign Idnetiy

* DID (Decentralized Identity)

* FIDO (Fast IDentity Online)
